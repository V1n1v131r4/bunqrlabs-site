Política de Privacidade

A BunqrLabs respeita a privacidade de seus visitantes e usuários. Este site foi projetado para minimizar coleta de dados e reduzir exposição desnecessária.

Coleta de dados

Não coletamos dados pessoais automaticamente. Informações só são fornecidas de forma voluntária, como em formulários de contato ou inscrição em listas.

Uso das informações

Dados fornecidos são utilizados exclusivamente para comunicação solicitada, envio de materiais ou resposta a contatos iniciados pelo próprio usuário.

Cookies

Este site não utiliza cookies de rastreamento comportamental ou publicidade. Serviços de terceiros podem aplicar suas próprias políticas.

Segurança

Adotamos práticas razoáveis para proteger informações, mas nenhum sistema é livre de riscos. O uso do site implica aceitação dessas limitações.

Contato

Para dúvidas sobre privacidade, entre em contato via contact@bunqrlabs.com.
